Recurring relapse triggers and cascade sequences documented from daily_signals.csv:

**Pattern 1: poker/alcohol cascade**
poker + beers → celebration mindset after win OR social weed offer → full relapse (lol + weed + clarity same night). Documented: 2026-03-06, 2026-02-04, 2026-01-31.

**Pattern 2: bad sleep → no gym → depletion → escape**
Late night → skip gym → willpower depleted → dopamine defaults. Documented: 2026-02-22, multiple recurrences.

**Pattern 3: overwhelm + binary trap**
Hard cognitive task (work, building) → depletion → brain presents only "hard thing vs League" — no middle option loaded. No pre-decided recharge activity = League wins by default. Documented: 2026-03-09 (reflection), 2026-03-29 (relapse).

**Pattern 4: "what's the point" ideation during hypofrontality**
After relapse or poor sleep, prefrontal cortex discredits sustained-effort projects (tracker, deep work). Strategic decisions made in this state are unreliable.

**Clarity streak fragility**
Clarity (no-fap) has never exceeded 4 days as of 2026-03-29. Frequent prolactin/dopamine crashes suppress motivation and amplify escape-seeking — this feeds League relapse more than the League addiction itself.

**Why:** Understanding these chains allows pattern-naming without re-reading full CSV history.
**How to apply:** When assessing brain state, check if current state matches any of these cascade entry points and name it directly.
