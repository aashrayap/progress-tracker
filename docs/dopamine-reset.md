## Gym Protocol (5x/week)

**Why 5x short vs 2-3x long:**
- Building the habit > optimizing gains
- Each visit = sauna meditation (habit stack)
- 5 meditation sessions/week vs 2-3
- Low friction: gym is 2 min away
- Easier to "just show up" for 1 hour

| Component | Duration |
|-----------|----------|
| Lift | 30-45 min |
| Sauna + meditation | 20 min |
| **Total** | **~1 hour** |

**Weekly Template:**

| Day | Workout |
|-----|---------|
| Mon | Day A (3-4 exercises) |
| Tue | Day B (3-4 exercises) |
| Wed | Day A |
| Thu | Day B |
| Fri | Day A |
| Sat | Rest or light cardio |
| Sun | Rest |

## What Replaces The Dopamine

```
OLD LOOP:                     NEW LOOP:
─────────────────────────     ─────────────────────────
Bored → LoL (5 hrs)           Bored → Gym / Walk / Book
Stressed → Weed               Stressed → Sauna / Walk
Evening → YouTube spiral      Evening → Cook / Read / Poker
Late night → Screens          Late night → Sleep earlier
```

**Replacement Activities:**
- Books (physical)
- Poker
- Walking
- Social (in person)
- Cooking
- Sleep earlier
